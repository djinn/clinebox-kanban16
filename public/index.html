Deploy the Kanban container

The Worker is running. To complete the setup, deploy the Kanban container separately:

# Build and push the container image GITHUB_TOKEN=ghp_xxx ./podman-build-and-push.sh
# Push container to Cloudflare npx wrangler container push --image ghcr.io/djinn/clinebox-kanban:latest --name clinebox-kanban-container --port 3484
# Update service binding and redeploy Worker npx wrangler deploy

Quick local test

Run Kanban locally to verify it works:

npx kanban --host 0.0.0.0
